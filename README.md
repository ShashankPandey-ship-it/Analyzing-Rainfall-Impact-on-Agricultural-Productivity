# 🌧️ Analyzing Rainfall Impact on Agricultural Productivity – Power BI Project

## 📊 Project Overview

This Power BI dashboard analyzes the relationship between rainfall patterns and agricultural productivity across different crops, states, and years.
The dashboard helps identify how rainfall variations influence crop yield, production volume, and cultivated area.

---

## 🎯 Business / Research Objective

The objective of this project is to:

* Understand the impact of rainfall on crop productivity
* Analyze yield trends over time
* Compare agricultural performance across states
* Identify optimal rainfall conditions for stable production
* Support data-driven agricultural planning and policy decisions
